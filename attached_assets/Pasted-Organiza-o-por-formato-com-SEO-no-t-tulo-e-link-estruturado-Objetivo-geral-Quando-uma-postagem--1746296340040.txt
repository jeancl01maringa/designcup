Organização por formato com SEO no título e link estruturado
📁 Objetivo geral:
Quando uma postagem for criada com múltiplos formatos (Feed, Cartaz, Stories, etc.), o Replit deve:

Criar um grupo de formatos vinculado (grupo_id)

Gerar títulos dinâmicos otimizados para SEO, incluindo o nome do formato

Armazenar links do Canva separados por formato

Associar cada imagem e cada link ao seu respectivo formato, tudo de forma organizada

🧠 Lógica Detalhada para Implementação
✅ 1. Título dinâmico com formato (SEO-friendly)
Durante a criação da arte:

O título base é inserido apenas uma vez na etapa 1 (ex: Flyer Tratamento de Pele)

O sistema automaticamente monta o título completo de cada arte ao salvar no Supabase, seguindo este padrão:

cpp
Copiar
Editar
Título final = [Título base] + " - Editável no Canva - " + [Formato]
Exemplos gerados:

Flyer Tratamento de Pele - Editável no Canva - Feed

Flyer Tratamento de Pele - Editável no Canva - Cartaz

Flyer Tratamento de Pele - Editável no Canva - Stories

Essa lógica pode rodar no backend, antes de salvar a arte.

🖼️ 2. Upload e link separados por formato
Cada formato selecionado (Feed, Cartaz, etc.) deve abrir uma aba específica para upload e link.

Para cada formato, o usuário deve:

Enviar a imagem

Inserir o link do Canva

Essas informações são armazenadas na tabela posts:

ts
Copiar

id: uuid
titulo: string (título completo com o formato)

licenca: string ('premium' ou 'free')
url_canva: string
url_imagem: string
🗃️ 3. Organização de links no Supabase
Armazenar cada arte com sua imagem no bucket por categoria, dentro de uma pasta com o formato como subdiretório:

Exemplo de estrutura no Supabase Storage:

swift
Copiar

/imagens/corporal/stories/123-stories.webp
🔁 4. Exibição dos formatos vinculados
Na página /arte/:id, deve ser possível:

Ver a arte principal

Abaixo, em “Outros formatos disponíveis”, listar todas as artes que compartilham o mesmo grupo_id

Cada miniatura deve exibir:

Imagem do formato correspondente

Botão "Editar no Canva" com o url_canva vinculado

✅ 5. Regras complementares
A URL do Canva e o upload da imagem são obrigatórios para cada formato

O botão de upload só libera o botão “Próximo” quando todos os formatos obrigatórios estiverem com imagem e link preenchidos

Quando a arte é exibida no feed ou na página de arte, o título completo com formato é exibido

⚙️ Resumo técnico para Supabase
Adicionar campo titulo_base (string)

Gerar titulo final no momento do insert com o formato incluído

Criar estrutura de bucket imagens/{categoria}/{formato}/

Armazenar url_imagem e url_canva separadamente por arte/registro

Agrupar por grupo_id e listar outros formatos na view da arte
